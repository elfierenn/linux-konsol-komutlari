# Linux Terminal Komutları
Bu projede 20 tane linux terminal komutları gösterilmiştir.

## Dizin İşlemleri (Navigasyon ve Yönetim)
#### pwd Komutu
Bulunduğunuz dizini gösterir.
```
Örnek:
> pwd
```
